# Project Euler 291
## 题目
### Panaitopol Primes


A prime number $p$ is called a Panaitopol prime if $p = \dfrac{x^4 - y^4}{x^3 + y^3}$ for some positive integers $x$ and $y$.

Find how many Panaitopol primes are less than 5\times10^15.
