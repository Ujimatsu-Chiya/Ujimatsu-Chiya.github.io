йўҳзӣ®

Writing 1/2 as a sum of inverse squares

There are several ways to write the number $\dfrac{1}{2}$ as a sum of inverse squares using distinct integers.

For instance, the numbers $\{2,3,4,5,7,12,15,20,28,35\}$ can be used:

In fact, only using integers between $2$ and $45$ inclusive, there are exactly three ways to do it, the remaining two being: $\{2,3,4,6,7,9,10,20,28,35,36,45\}$ and $\{2,3,4,6,7,9,12,15,28,30,35,36,45\}$.

How many ways are there to write the number $\dfrac{1}{2}$ as a sum of inverse squares using distinct integers between $2$ and $80$ inclusive?

и§ЈеҶіж–№жЎҲ

жң¬и§ЈеҶіж–№жЎҲеҸӮиҖғдәҶThreadдёӯзҡ„дёҖдәӣеҶ…е®№гҖӮ

дёәеҮҸе°‘жһҡдёҫйҮҸпјҢиҝӣиЎҢд»ҘдёӢжҺ’йҷӨпјҡ

  1. иҖғиҷ‘жүҖжңүиҙЁж•°$p$пјҢд»Ө$q=p^r$пјҢе…¶дёӯ$r$жҳҜдҪҝ$p^r$дёҚи¶…иҝҮ$N=80$зҡ„жңҖеӨ§еҖјгҖӮ

еңЁиҝҷз§Қжғ…еҶөдёӢпјҢиҖғиҷ‘жүҖжңүж»Ўи¶і$kq\le N$зҡ„еҲҶж•°$\dfrac{1}{q^2},\dfrac{1}{(2q)^2},\dots,\dfrac{1}{(kq)^2}$

иҖғиҷ‘и®Ўз®—$\dfrac{1}{(i_1q)^2}+\dfrac{1}{(i_2q)^2}+\dots+\dfrac{1}{(i_mq)^2}$пјҢе…¶дёӯ$i_j<p$пјҢдёҚеҗҢзҡ„$i_j$д№Ӣй—ҙдёӨдёӨдёҚеҗҢгҖӮ

йӮЈд№Ҳжңү$\dfrac{1}{q^2}(\dfrac{1}{i_1^2}+\dfrac{1}{i_2^2}+\dots+\dfrac{1}{i_m^2})$

еҰӮжһңеңЁи®Ўз®—еҲҶж•°е’Ңзҡ„иҝҮзЁӢдёӯпјҢеҲҶжҜҚж··иҝӣдәҶдёҖдёӘйқһ$2$зҡ„иҙЁж•°$p$пјҢйӮЈд№ҲеңЁд»ҘеҗҺжұӮе’Ңзҡ„иҝҮзЁӢдёӯеҝ…йЎ»ж¶ҲйҷӨеҲҶжҜҚ$p$гҖӮеӣ жӯӨ$\sum_{j=1}^m i_j^{-2}$еңЁжЁЎ$q^2$дёҠдёҺ$0$еҗҢдҪҷгҖӮ

  • еҜ№дәҺеӨ§дәҺ$\dfrac{N}{2}$зҡ„иҙЁж•°иҖҢиЁҖпјҢеҸӘжңүдёҖдёӘ$m=1,i_1=1$пјҢд№ҹе°ұжҳҜ$\{1\}$пјҢдёҚеҸҜиғҪеӯҳеңЁдёҖдёӘйқһз©әйӣҶеҗҲдҪҝеҫ—е…¶еӯҗйӣҶе’Ңдёә$0$гҖӮж•…жҺ’йҷӨиҝҷдёҖзұ»жғ…еҶөгҖӮ

  • еҜ№дәҺиҙЁж•°$p=11$пјҢйӮЈд№Ҳ$i_j\in[1,7]$пјҢд№ҹе°ұжҳҜиҜҙпјҢ$i_j^{-2}\% p^2 \in\{1, 91, 27, 53, 92, 37, 42\}$пјҢе®№жҳ“з»ҸиҝҮйӘҢиҜҒпјҢжІЎжңүдёҖдёӘйқһз©әеӯҗйӣҶзҡ„е’Ңдёә$p^2$пјҢж•…жҺ’йҷӨ$11$зҡ„жүҖжңүеҖҚж•°гҖӮ

  • еҜ№дәҺиҙЁж•°$p=13$пјҢйӮЈд№Ҳ$i_j\in[1,6]$пјҢд№ҹе°ұжҳҜиҜҙпјҢ$i_j^{-2}\% p^2 \in\{1, 127, 94, 74, 142, 108\}$пјҢе®№жҳ“з»ҸиҝҮйӘҢиҜҒпјҢеҸӘжңү$1+94+74=169$жүҚжҳҜ$p^2$зҡ„еҖҚж•°пјҢеӣ жӯӨйҷӨдәҶ$13,39,52$пјҢе…¶е®ғйғҪеҸҜд»ҘжҺ’йҷӨгҖӮиҖҢдё”пјҢиҝҷдёүдёӘж•°еҸӘжңүдёҖиө·з”ЁжүҚиғҪж¶ҲйҷӨжҺүеҲҶжҜҚ$13$гҖӮз»Ҹи®Ўз®—еҫ—$\dfrac{1}{13^2}+\dfrac{1}{39^2}+\dfrac{1}{52^2}=\dfrac{1}{12^2}$пјҢжүҖд»ҘеҸҜд»ҘеңЁеҖҷйҖүйӣҶдёӯеӨҡж·»еҠ дёҖдёӘ$12$гҖӮз„¶еҗҺеҝҪз•Ҙиҝҷ$3$дёӘж•°гҖӮ

еҸҜд»Ҙз”Ёзұ»дјјзҡ„ж–№жі•пјҢжҺ’йҷӨжҺүд»ҘдёӢиҝҷдәӣж•°пјҡ$\{17,34,51,68,19,38,57,76,23,46,69,27,54,29,58,49,64,вҖҰ\}$

жӣҙиҝӣдёҖжӯҘпјҢеҰӮжһң$q=p^r$дёӯзҡ„$r$дёҚдёҖе®ҡжҳҜж»Ўи¶іжңҖеӨ§еҖјпјҢдёҠиҝ°$i_j$д№ҹеҸӘжҳҜж»Ўи¶і$\gcd(i_j,q)=1$пјҢйӮЈд№Ҳз”Ёзұ»дјјзҡ„ж–№жі•пјҢеҸҜд»Ҙе°Ҷ$\{16,32,48,80\}$жҺ’йҷӨгҖӮ

  1. ж №жҚ®дёҠйқўзҡ„жҺ’йҷӨз»“жһңпјҢеү©дёӢзҡ„ж•°дёӯпјҢ$\dfrac{1}{2^2},\dfrac{1}{3^2}$дёҖе®ҡжҳҜеңЁйӣҶеҗҲдёӯзҡ„пјҢеӣ дёәзјәе°‘дәҶиҝҷдёӨдёӘж•°д№ӢдёҖпјҢеү©дёӢзҡ„еҲҶж•°зҡ„е’Ңе°ҸдәҺ$\dfrac{1}{2}$гҖӮеӣ жӯӨдёҚйңҖиҰҒеҜ№иҝҷдёӨдёӘж•°жһҡдёҫпјҢеҸӘиҰҒеңЁжңҖз»Ҳе’ҢеҖј$\dfrac{1}{2}$дёӯеҮҸеҺ»еҚіеҸҜгҖӮ

жҢүз…§дёҠйқўзҡ„ж–№жі•пјҢжңҖз»ҲеҸӘйңҖиҰҒжһҡдёҫ$27$дёӘж•°зҡ„з»„еҗҲжғ…еҶөгҖӮ

иҝҷйҮҢеҲҷдҪҝз”Ёmeet-in-the-middleжҖқжғіпјҡе…Ҳе°ҶеҲҶж•°еҲҶжҲҗе°ҪйҮҸзӣёеҗҢеӨ§е°Ҹзҡ„дёӨйғЁеҲҶпјҢз„¶еҗҺеӯҗйӣҶжһҡдёҫз¬¬дёҖйғЁеҲҶпјҢе№¶жҢүеӯҗйӣҶе’Ң$s$е°Ҷз»“жһңеӯҳеӮЁиө·жқҘпјҢз„¶еҗҺеӯҗйӣҶжһҡдёҫз¬¬дәҢйғЁеҲҶпјҢжұӮеҮә$sвҖҷ$еҗҺпјҢеңЁз¬¬дёҖйғЁеҲҶзҡ„жһҡдёҫз»“жһңдёӯжҹҘжүҫжңүжІЎжңүйӮЈзјәеӨұзҡ„$s$пјҢдҪҝеҫ—$s+sвҖҷ$жҳҜеҺҹжқҘзҡ„й—®йўҳзҡ„и§ЈгҖӮеҰӮжһңжңүпјҢйӮЈе°ұжүҫеҲ°дәҶи§ЈгҖӮ
