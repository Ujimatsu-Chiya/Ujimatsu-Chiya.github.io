Convex Holes

Given a set of points on a plane, we define a convex hole to be a convex polygon having as vertices any of the given points and not containing any of the given points in its interior (in addition to the vertices, other given points may lie on the perimeter of the polygon).

As an example, the image below shows a set of twenty points and a few such convex holes. The convex hole shown as a red heptagon has an area equal to $1049694.5$ square units, which is the highest possible area for a convex hole on the given set of points.

For our example, we used the first $20$ points $(T_{2kвҲ’1},T_{2k})$, for $k=1,2,\dots,20$, produced with the pseudo-random number generator:

$\begin{aligned}
S_0&=290797\\
S_{n+1}&=S_n^2 \text{ mod } 50515093\\
T_n&=(S_n \mod 2000)-1000
\end{aligned}$

i.e. $(527,144), (вҲ’488,732), (вҲ’454,вҲ’947), \dots$

What is the maximum area for a convex hole on the set containing the first $500$ points in the pseudo-random sequence?

Specify your answer including one digit after the decimal point.

и§ЈеҶіж–№жЎҲ

жң¬йўҳе°Ҷе°қиҜ•жүҫеҲ°жүҖжңүзҡ„еҮёз©әжҙһпјҢе№¶дё”жұӮеҮәе®ғд»¬зҡ„жңҖеӨ§еҖјгҖӮдёәдәҶеҮҸе°‘зЁӢеәҸиҝҗиЎҢж—¶й—ҙпјҢжҲ‘д»¬е°ҪйҮҸеҮҸе°‘еҮёз©әжҙһзҡ„йҮҚеӨҚеҜ»жүҫж¬Ўж•°пјҢйҮҮеҸ–зҡ„жҺӘж–ҪжҳҜпјҡжһҡдёҫеӣҫдёӯзҡ„жҜҸдёҖдёӘзӮ№$A(x_i,y_i)$дҪңдёәж•ҙдёӘеқҗж Үзі»зҡ„еҺҹзӮ№пјҢе°Ҷж»Ўи¶іиҝҷдәӣжқЎд»¶зҡ„зӮ№$(x_j,y_j)$иҲҚеҺ»пјҡ$y_j<y_i\vee (y_j=y_i \wedge x_j<x_i)$.йӮЈд№Ҳеү©дёӢзҡ„иҝҷдәӣзӮ№дёӯпјҢ$A$жҳҜжңҖдёӢж–№зҡ„пјҢиҖғиҷ‘д»Ҙ$A$дёәз©әжҙһзҡ„дёҖдёӘзү№е®ҡзӮ№иҝӣиЎҢжһҡдёҫгҖӮ

йӮЈд№ҲпјҢе°ҶжүҖжңүзҡ„зӮ№йғҪжҢүз…§жһҒи§’иҝӣиЎҢжҺ’еәҸгҖӮиҖғиҷ‘еҰӮдёӢеӣҫзҡ„еҪўејҸпјҢдё»иҰҒжҖқжғіжҳҜд»ҺйҖҶж—¶й’Ҳж–№еҗ‘иө·жүҫеҲ°дёҖдёӘдёӘдёүи§’еҪўпјҢиҝҷдәӣдёүи§’еҪўеҗҲе№¶еҗҺжҒ°еҘҪеҪўжҲҗдёҖдёӘеҮёеҢ…гҖӮ

еҒҮи®ҫеҪ“еүҚдёҖе…ұжңү$m$дёӘзӮ№пјҢиҝҷдәӣзӮ№жһҒи§’жҺ’еәҸеҗҺпјҢеҲҶеҲ«ж Үи®°жҲҗ$B_1,B_2,\dots,B_m$гҖӮ

иҖғиҷ‘зҠ¶жҖҒ$f(i,j)(1\le j< i\le m)$иЎЁзӨәйҖҶж—¶й’Ҳж–№еҗ‘дёҠпјҢд»Ҙ$\triangle AB_iB_j$дёәжңҖеҗҺдёҖдёӘдёүи§’еҪўзҡ„еҮёеҢ…дёӯпјҢжңҖеӨ§зҡ„йқўз§ҜеҖјгҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢдёҚеӯҳеңЁе…¶е®ғзӮ№$B_k$дҪҝеҫ—$B_k$иҗҪеңЁ$\triangle AB_iB_j$зҡ„еҶ…йғЁпјҲдёҚеҢ…жӢ¬иҫ№з•Ңпјүж—¶пјҢе№¶дё”$O,B_i,B_j$дёҚе…ұзәҝж—¶пјҢ$f(i,j)$жүҚжҳҜдёҖдёӘеҗҲжі•зҠ¶жҖҒгҖӮеҜ№дәҺжүҖжңүдёҚеҗҲжі•зҡ„зҠ¶жҖҒпјҢзҠ¶жҖҒеҖј$f$дёә$0$гҖӮ

йӮЈд№ҲпјҢ$f$зҡ„зҠ¶жҖҒиҪ¬з§»ж–№зЁӢе°ұеҸҜд»ҘеҶҷжҲҗеҰӮдёӢеҪўејҸпјҡ

еҸӘиҰҒзӮ№$B_k$еңЁеҗ‘йҮҸ$\overrightarrow{B_jB_i}$зҡ„е·Ұдҫ§пјҢд»ҺиҖҢ$B_i,B_j,B_k$еҪўжҲҗдёҖдёӘеҮёеҪўгҖӮйӮЈд№ҲзҠ¶жҖҒ$f(j,k)$жүҖд»ЈиЎЁзҡ„еҮёеҢ…е’Ң$\triangle AB_iB_j$еҗҲе№¶еҪўжҲҗдәҶдёҖдёӘж–°еҮёеҢ…гҖӮ

еңЁжһҡдёҫ$i$ж—¶пјҢйңҖиҰҒе…ҲйЎәж—¶й’ҲжүҫеҲ°з¬¬дёҖдёӘ$j$пјҢдҪҝеҫ—$O,B_i,B_j$е…ҲдёҚе…ұзәҝгҖӮз„¶еҗҺжүҫеҲ°жүҖжңүзҡ„$k$дҪҝеҫ—$\overrightarrow{B_jB_i}\times\overrightarrow{B_jB_k}>0$пјҢз„¶еҗҺзӣҙжҺҘиҝӣиЎҢиҪ¬з§»гҖӮеҰӮжһңзӣҙжҺҘиҝҷж ·еӯҗе®һзҺ°пјҢйӮЈд№ҲжҜҸж¬ЎжһҡдёҫдёҖдёӘеҺҹзӮ№пјҢе°ұйңҖиҰҒ$O(n^3)$зҡ„иҪ¬з§»пјҢжҖ»дҪ“ж—¶й—ҙеӨҚжқӮеәҰдёә$O(n^4)$.

иҖғиҷ‘еҰӮжӯӨиҝӣиЎҢпјҡеҰӮжһңжҲ‘д»¬е·Із»ҸзЎ®е®ҡдәҶ$f(i,j)$жҳҜдёҖдёӘеҗҲжі•зҠ¶жҖҒпјҢйӮЈд№ҲжҺҘдёӢжқҘе°ұеҺ»еҜ»жүҫ$k$пјҢдҪҝеҫ—$f(j,k)$д№ҹжҳҜдёҖдёӘеҗҲжі•зҠ¶жҖҒгҖӮеҰӮжһңжүҫеҲ°дёҖдёӘжңҖеӨ§зҡ„$k_0$дҪҝеҫ—$\overrightarrow{B_jB_i}\times\overrightarrow{B_jB_{k_0}}>0$пјҢйӮЈд№ҲеҜ№дәҺ$k=k_0+1,k_0+2,\dots,j-1,\overrightarrow{B_jB_i}\times\overrightarrow{B_jB_{k}}<0$йғҪжҲҗз«ӢпјҢд№ҹе°ұжҳҜиҜҙпјҢ$B_j$еңЁ$\triangle OB_iB_k$еҶ…йғЁпјҢйӮЈд№Ҳ$f(i,k)$йғҪдёҚжҳҜеҗҲжі•зҡ„пјҢе…¶йғҪеҖјдёә$0$пјҢеҰӮдёӢеӣҫжүҖзӨәгҖӮ

еҜ№дәҺжүҖжңүзҠ¶жҖҒ$1\le kвҖҷ\le k_0$пјҢжңүд»ҘдёӢиҜҙжі•пјҡ

  1. $B_{kвҖҷ}$еңЁеҗ‘йҮҸ$\overrightarrow{B_iB_{j}}$е·Ұдҫ§пјҢе®ғд»¬йғҪжҳҜдёҚеҗҲжі•зҡ„пјҢеӣ дёә$B_j$еңЁ$\triangle AB_iB_k$еҶ…йғЁгҖӮ
  2. е…¶дҪҷжғ…еҶөйғҪеҸҜд»ҘиҝӣиЎҢиҪ¬з§»гҖӮ

йӮЈд№ҲзҠ¶жҖҒиҪ¬з§»ж–№зЁӢе°ұеҸҜд»Ҙж”№еҶҷжҲҗпјҡ

йҖҡиҝҮж•°жҚ®з»“жһ„з»ҙжҠӨпјҢиҝҷдёӘйҖ’жҺЁејҸеҸҜд»ҘйҖҡиҝҮ$O(n^2)$жұӮеҮәжқҘгҖӮ

иҝҷдёӘжҖқи·ҜиҝҳжҸҗдҫӣдәҶдёҖз§Қеҝ«йҖҹжүҫеҲ°еҗҲжі•зҠ¶жҖҒзҡ„ж–№жЎҲпјҡеҜ№дәҺдёҖдёӘзҠ¶жҖҒ$f(i,j)$пјҢи®Ўз®—еҮәжқҘдәҶ$k_0$еҗҺпјҢиҝҷдёӘ$k_0$е°ҶдҪңдёәдёӢдёҖж¬ЎжұӮи§Јзҡ„$j$гҖӮ

еҸҰеӨ–пјҢеҜ№дәҺд»»ж„ҸдёӨзӮ№$B_i,B_j,S_{\triangle AB_i,B_j}=|\overrightarrow{AB_i}\times\overrightarrow{AB_j}|$.

жңҖз»Ҳзӯ”жЎҲдёә$\max_{i=1}^m\{\max_{j=1}^if(i,j)\}$.
